This site tries to gather of great old text mode games in one place. If you think that something is missing in the list - please go to our Github repository and create an issue or even a pull request!

Similar resources: check out TTY Games, which specializes on games in text mode, Linux Game Database, contains a list of ASCII games, and Open Source Game Clones.

Type "play" to find all games available on remote servers via telnet or ssh. Type "screencast" to find all games with recorded screencasts.
